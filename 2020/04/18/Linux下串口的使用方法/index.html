Linux下串口的使用方法

安装minicom

  • 设置minicom

  • 打开一个shell终端, 执行sudo minicom
  • 按下Ctrl+a, 然后按下z, 弹出帮助菜单, 按下o, 通过上下箭头按键选择Serial port setup, 进入设置串口的界面, 可以设置要打开的串口设备, 波特率等
  • Save setup as dfl 把刚才的设置保存为默认设置
  • Add Carriage Ret 表示换行时自动回车(从第一位开始打印)
