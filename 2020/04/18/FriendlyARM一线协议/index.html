FriendlyARM一线协议

主板和LCD驱动板通过一线协议通讯, 实现电阻触摸, 背光控制和识别型号等功能.

读写

  • host发命令给client，格式为8bit cmd + 8bit crc。
  • client返回16bit data + 8bit crc。

命令

  • 发送0x80~0xff给client，直接设置背光。

信号

  • 旧版本协议使用一根GPIO。
  • 新版本使用一路I2C和一根GPIO。
